Hubungkan ke Firebase untuk

Langkah Setup
1
Buka console.firebase.google.com, buat project baru (gratis)
2
Project Settings → tambah Web App → salin konfigurasi
3
Realtime Database → Create database → mode test
4
Tempel konfigurasi di bawah lalu klik Simpan
ℹ️ Konfigurasi disimpan di browser. Bagikan file HTML ini ke semua pengguna dengan konfigurasi yang sama. Data sinkron real-time otomatis.
